```bash
terragrunt scaffold <MODULE_URL> [TEMPLATE_URL] [--var] [--var-file] [--no-include-root] [--root-file-name]
```

For more information on how scaffolding works, see the dedicated [scaffold documentation](/docs/features/scaffold).
