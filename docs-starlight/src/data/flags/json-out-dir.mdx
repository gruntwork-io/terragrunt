---
name: json-out-dir
description: Directory to store JSON plan files generated by plan/show commands.
type: string
env:
  - TG_JSON_OUT_DIR
---

Specifies the directory where Terragrunt will write OpenTofu/Terraform plan files in JSON format. Useful for automation and tooling that parses machine-readable plans.

If the directory does not exist, Terragrunt will attempt to create it.

See examples in the Stacks feature docs.
