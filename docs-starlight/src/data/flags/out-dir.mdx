---
name: out-dir
description: Directory to store plan files generated by plan/apply/show commands.
type: string
env:
  - TG_OUT_DIR
---

Specifies the directory where Terragrunt will write OpenTofu/Terraform plan files when running stack operations, e.g. `plan`, `apply`, or `show`.

If the directory does not exist, Terragrunt will attempt to create it.

See examples in the Stacks feature docs.
