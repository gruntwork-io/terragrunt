---
name: provider-cache-hostname
description: The hostname of the Terragrunt Provider Cache server. By default, 'localhost'.
type: string
env:
  - TG_PROVIDER_CACHE_HOSTNAME
---

Sets the hostname for connecting to the Provider Cache server. This flag is only used when [`provider-cache`](#provider-cache) is enabled.

To learn more about how to use this flag, see the [Provider Cache Server](/docs/features/provider-cache-server) feature documentation.
