#### Discoverability of `init-from-module` documentation improved

The special internal `init-from-module` command referenced in hooks has had its documentation improved to make it easier to discover. It was difficult to find in the [terraform HCL block](/reference/hcl/blocks/#terraform) documentation, and that resulted in confusion for users.
