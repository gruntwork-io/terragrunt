#### 1.0 Guarantees

A living document named [Terragrunt 1.0 Guarantees](/process/1-0-guarantees/) has been added to the Terragrunt website clarifying what is and isn't considered a breaking change for the duration of 1.0.

Over time, as ambiguity in edge-cases for what is considered a breaking change are addressed, the page will be updated so that you can be confident your workflows won't be impaired.
