#### Tip builds now available from `main`

Every successful CI run on the `main` branch now automatically produces tip build binaries with signed checksums for all supported platforms (Windows and macOS binaries are not codesigned in tip builds). These builds are accessible via the builds API at `https://builds.terragrunt.com` — see the [releases process documentation](/process/releases#tip-builds) for API endpoints and usage examples. Maintainers can also trigger on-demand test builds from any branch using the `test-build.yml` workflow.
