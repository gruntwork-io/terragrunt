#### Fixed concurrent write conflict in `terragrunt stack generate`

Fixed non-deterministic file errors during `terragrunt stack generate` on nested stack hierarchies. The dedup map treated string-aliased paths to the same physical file as distinct, so the same file could be dispatched twice and the duplicate writes raced. Stack-file paths are now cleaned, made absolute at discovery, and resolved through symlinks (best-effort), so aliased forms collapse to a single dedup entry.
