#### Friendly panic crash reporting with crash logs

Terragrunt now writes a `terragrunt-crash-*.log` file with full panic context instead of dumping a raw stack trace. The terminal shows a short banner pointing to the file and to https://github.com/gruntwork-io/terragrunt/issues. If the log can't be written, the report is printed inline.
