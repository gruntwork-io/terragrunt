#### Friendly panic reports

On a panic, Terragrunt now writes a `terragrunt-crash-<UTC-timestamp>-<pid>.log` file in the working directory (falling back to the system temp directory) with full runtime context, then logs the file path and the bug-report URL: https://github.com/gruntwork-io/terragrunt/issues. The raw stack trace no longer floods the terminal. If the file cannot be written, the same content is printed inline so nothing is lost.
