#### Friendly panic crash reporting with crash logs

Terragrunt now provides detailed crash reports when unrecoverable panics occur.

When a panic is captured, Terragrunt writes a timestamped `terragrunt-crash-*.log` file containing runtime context, command-line details, Terragrunt version, and the full stack trace. The terminal output now points users to that file and requests they report the issue at:

https://github.com/gruntwork-io/terragrunt/issues
