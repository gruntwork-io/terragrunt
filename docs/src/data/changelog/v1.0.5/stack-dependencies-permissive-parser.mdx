#### `stack-dependencies`: phased parser for `terragrunt.stack.hcl`

`terragrunt.stack.hcl` is now parsed in four phases: parse root -> locals -> includes -> unit/stack decode; it supports `local.*`, `values.*`, and terragrunt functions, and run-all discovery still uses path-only nested traversal while unifying strict and permissive parser logic.
