#### Friendly panic reports

On a panic, Terragrunt writes a `terragrunt-crash-<UTC-timestamp>-<pid>.log` file with runtime context, the panic message, and the stack trace.

The terminal output shows where the report was saved, the Terragrunt version, and the bug-report URL. If Terragrunt cannot write the file, it prints the report inline so the diagnostic details are not lost.
