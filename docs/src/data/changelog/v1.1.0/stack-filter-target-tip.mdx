#### Tip when filtering a stack that contains nested stacks

`terragrunt stack generate --filter './my-stack | type=stack'` generates only the selected
stack, not the nested stacks it contains, which can be surprising for a stack of stacks.
When a non-glob `| type=stack` filter targets a stack whose generated directory contains
nested stacks, Terragrunt now prints a tip showing how to generate them too, for example
`--filter './my-stack/** | type=stack'`.
