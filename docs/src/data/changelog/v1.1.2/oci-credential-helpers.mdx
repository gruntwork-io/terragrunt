#### `oci` - Credential helpers for OCI module sources

`oci://` module downloads now honor the Docker credential helpers configured in your Docker config: per-registry `credHelpers` and the global `credsStore`, invoking `docker-credential-<helper>` (including `ecr-login`). Credentials are re-minted on every run, so registries with short-lived tokens such as Amazon ECR work without a baked-in login. Helpers run under Terragrunt's effective environment, so an assumed AWS role reaches `ecr-login`. No new configuration is required beyond the standard Docker config.
