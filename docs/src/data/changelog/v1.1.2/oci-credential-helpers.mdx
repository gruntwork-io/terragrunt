#### `oci` - Credential helpers for OCI module sources

Downloading a module from an `oci://` source now reuses the Docker credential helpers you already have configured. Registries that hand out short-lived login tokens, such as Amazon ECR, now authenticate automatically, and there is nothing new to set up: if `docker pull` works against a registry, Terragrunt can pull modules from it too.
