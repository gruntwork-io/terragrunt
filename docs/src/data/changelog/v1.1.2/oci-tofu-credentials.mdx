#### `oci` - OpenTofu CLI-config credentials for OCI module sources

`oci://` module downloads now read OpenTofu's CLI-config credentials, so one configuration serves both `tofu` and Terragrunt. Terragrunt honors the `oci_credentials "<registry>/<repo>"` blocks (username/password, OAuth tokens, or a `docker_credentials_helper`) and the `oci_default_credentials` fallback helper from your `.tofurc` / `.terraformrc` (or the file named by `TF_CLI_CONFIG_FILE`). Per-registry `oci_credentials` blocks take precedence over ambient Docker-config discovery, and the `oci_default_credentials` helper is a lower-priority fallback; `discover_ambient_credentials = false` disables ambient discovery. The read is best-effort: a missing or unparsable CLI config is skipped rather than failing the download.
