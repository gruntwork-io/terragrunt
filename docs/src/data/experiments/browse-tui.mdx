Adds the `terragrunt browse` command, which browses the discovered infrastructure estate in an interactive terminal user interface (TUI).

### `browse-tui` - What it does

With the experiment enabled, `terragrunt browse` opens a [yazi](https://yazi-rs.github.io/)-style, three-column browser of the discovered estate. See the [`browse` command documentation](/reference/cli/commands/browse) for usage and keybindings.

Running `terragrunt browse` without the experiment enabled is an error.
