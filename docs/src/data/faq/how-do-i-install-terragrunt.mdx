There are several ways to install Terragrunt:

- Using a package manager such as [Homebrew](https://brew.sh/) (macOS/Linux) or [Chocolatey](https://chocolatey.org/) (Windows).
- Downloading a prebuilt binary from the [GitHub releases page](https://github.com/gruntwork-io/terragrunt/releases).
- Running the official install script.

You also need a supported version of [OpenTofu](https://opentofu.org/) or [Terraform](https://developer.hashicorp.com/terraform) installed.

For step-by-step instructions for each method, see the [Install](/getting-started/install/) page.
