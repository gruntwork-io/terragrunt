Terragrunt is a flexible orchestration tool that allows Infrastructure as Code written in [OpenTofu/Terraform](https://opentofu.org/) to scale.

It wraps OpenTofu/Terraform and adds tooling for:

- Keeping your OpenTofu/Terraform configurations [DRY](/features/units/) (Don't Repeat Yourself).
- Managing [dependencies](/features/stacks/) between units of infrastructure.
- Running commands across many units at once.
- Keeping your remote state and provider configuration consistent across your codebase.

To learn more, read the [Overview](/getting-started/overview/) and try the [Quick Start](/getting-started/quick-start/).
