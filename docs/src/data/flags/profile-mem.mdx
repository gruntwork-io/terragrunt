import { Aside } from '@astrojs/starlight/components';

<Aside type="tip" title="Experimental">
This flag is gated behind the [`pprof`](/reference/experiments/active#pprof) experiment. Enable it with `--experiment=pprof` or `TG_EXPERIMENT=pprof`.
</Aside>

Write a memory (heap) profile of the Terragrunt process at exit (after forcing a GC).

This is the Terragrunt process only — there is no equivalent for downstream OpenTofu/Terraform.

Analyze with:

```shell
go tool pprof terragrunt_mem.prof
```

When using `--profile-dir`, the file is written as `terragrunt_mem.prof` inside the directory.
